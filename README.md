# ConoHa Puzzle
こちらからゲームが遊べます。  
[Github Pages](https://tacky0612.github.io/ConoHaPuzzle/)
 * * *  
## 【ゲーム説明】

* 計16枚のパネルが入るエリアと15枚のイメージをスライドさせて正しい並びを目指すパズルです。
* 【ゲーム開始】ボタンでゲームを始めることが出来ます。
* パズルが完成したら【完成！】ボタンで結果が表示されます。
* より短い時間・より少ないクリック数で完成を目指しましょう！！

 * * *
